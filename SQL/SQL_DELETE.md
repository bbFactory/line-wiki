# SQL

## DELETE 문

  - `DELETE` 문은 테이블의 기존 레코드를 삭제하기 위해 사용한다.

  - ( 주의! )  **`WHERE` 절** 을 생략하면 테이블의 모든 레코드가 삭제된다.

    > **DELETE FROM**  table_name
    >
    > **WHERE**  condition;
    >
    > (실사용 예시)
    >
    > **DELETE FROM** Customers 
    >
    > **WHERE** CustomerName='Peter Parker';
