# SQL

## SQL MIN  /  MAX 함수

  - `MIN()` 함수는 선택한 열의 가장 작은 값을 반환한다.

    > **SELECT MIN**(column_name)
    >
    > **FROM** table_name
    >
    > **WHERE** condition;

  - `MAX()` 함수는 선택한 열의 가장 큰 값을 반환한다.

    > **SELECT MAX**(column_name)
    >
    > **FROM** table_name
    >
    > **WHERE** condition;
