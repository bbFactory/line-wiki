# Rule of 1.01 ( 1.01의 법칙 )

- **1.01**은 제곱할수록 점점 더 커지지만, &nbsp; **0.99**는 제곱을 할수록 점점 더 작아집니다.
- **1.00**은 아무리 제곱해도 1.00 그대로입니다.
- 당장의 1% 또는 2% 차이는 크게 느껴지지 않습니다. 하지만 **3년, 5년**, 아니 **10년 후**라면 어떨까요?
